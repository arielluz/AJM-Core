# AJM-Core
![AJM-Core](img/banner.png)

O **AJM-Core** é um sistema de monitoramento paranormal desenvolvido para reconstruir os arquivos perdidos da Ordem.

A plataforma permite catalogar entidades, registrar ocorrências e organizar informações classificadas de acordo com níveis de ameaça.

Inspirado no universo de **Ordem Paranormal**, o projeto apresenta uma interface de investigação contendo arquivos, registros e descrições de criaturas.

---

# 📌 Funcionalidades

 Catálogo de Entidades

- Visualização de criaturas cadastradas.
- Organização por níveis de ameaça.
- Acesso aos arquivos individuais de cada entidade.

 Sistema de Classificação

As entidades são divididas em três níveis:
 Nível de Ameaça 1
Entidades de baixo risco.


 Nível de Ameaça 2
Entidades perigosas.

  Nível de Ameaça 3
Entidades extremamente perigosas
